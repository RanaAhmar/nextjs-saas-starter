## ✨ Overview

Stop configuring Webpack, setting up authentication, and fighting with Stripe forms every time you have an idea. 

The **Next.js 15 SaaS Starter** provides a robust, scalable foundation built on the App Router, featuring enterprise-grade authentication, multi-tenant billing, and beautifully designed UI components out of the box.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router, Server Actions)
- **Styling**: Tailwind CSS & shadcn/ui components
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js (v5) with Magic Links & OAuth
- **Payments**: Stripe Checkout & Customer Portal integration
- **Emails**: Resend & React Email

## 📦 Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/RanaAhmar/nextjs-saas-starter.git my-saas
cd my-saas

# 2. Install dependencies
npm install

# 3. Setup environment
cp .env.example .env.local

# 4. Push database schema
npx prisma db push

# 5. Start development server
npm run dev
```
